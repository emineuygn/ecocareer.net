# Echo AI

Echo AI, bir hackathon kapsamında geliştirilmiş, kullanıcılarla etkileşimli şekilde çalışan
AI destekli bir uygulamadır.  
Proje, kullanıcı girdilerini analiz ederek anlamlı ve kişiselleştirilmiş yanıtlar üretmeyi amaçlar.
Bu proje bir **hackathon** sürecinde, kısa sürede çalışır bir prototip ortaya koymak amacıyla geliştirilmiştir.
 Özellikler
- 🤖Yapay zeka destekli etkileşim
- 🧠Kullanıcı girdilerine dayalı yanıt üretimi
- 🎨 Basit ve kullanıcı dostu arayüz
- 📁 JSON tabanlı veri yönetimi
- 🖥️ Masaüstü / uygulama tabanlı yapı

 Kullanılan Teknolojiler
- **Python**
- **Kivy (kvlang)**
- **JSON**
- Yapay zeka tabanlı karar mekanizması

⚙️ Kurulum
```bash
git clone https://github.com/Deryaakscc/echo-ai.git
